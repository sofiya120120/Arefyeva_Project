# Система автоматической генерации мультимодальных образовательных

![Python](https://img.shields.io/badge/Python-3.9%2B-blue)
![FastAPI](https://img.shields.io/badge/FastAPI-0.104%2B-green)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28%2B-red)


**Умное приложение для создания учебных карточек вопрос-ответ с искусственным интеллектом**

## Возможности

-  **Анализ текста** - автоматическое выделение ключевых предложений
-  **Генерация вопросов** - ИИ создает вопросы на основе текста
-  **Создание изображений** - генерация тематических фонов с помощью FLUX AI
-  **Красивые карточки** - профессиональное оформление вопросов и ответов
-  **Экспорт** - возможность скачать карточки в ZIP архиве

##  Для кого этот проект?

-  **Студенты** - для создания учебных материалов
-  **Преподаватели** - для подготовки занятий
-  **Самообучающиеся** - для систематизации знаний
-  **Компании** - для обучения сотрудников

## Технологии

- **Backend**: FastAPI + Python
- **Frontend**: Streamlit
- **ИИ модели**:
  - FLUX.1-schnell (генерация изображений)
  - T5-base (генерация вопросов)
  - Google Translate API (перевод)
- **Обработка изображений**: Pillow

##  Быстрый старт

### 1. Клонирование репозитория

```bash
git clone https://github.com/ваш-username/smartflash-ai.git
cd smartflash-ai
```
### 2. Установка зависимостей
```bash
pip install -r requirements.txt
```
### 3. Настройка окружения
Создай файл .env на основе .env.example:

```bash
cp .env.example .env
```
Добавь в .env свой HuggingFace токен:
```bash
env
HF_TOKEN=your_huggingface_token_here
```
### 4. Запуск backend сервера
```bash
cd backend
uvicorn main:app --reload --host 0.0.0.0 --port 8000
```
#### Сервер будет доступен по адресу: http://localhost:8000

### 5. Запуск frontend интерфейса
```bash
cd frontend
streamlit run app.py
```
#### Интерфейс будет доступен по адресу: http://localhost:8501

### Конфигурация
| Переменная | Описание | Пример |
| :--- | :--- | :---: |
| **`HF_TOKEN`** | Токен HuggingFace для доступа к моделям генерации изображений | `hf_xxxxxxxxxxxxxxxx` |
| `API_URL` | URL для доступа к модели FLUX.1-schnell | `https://router.huggingface.co/hf-inference/models/black-forest-labs/FLUX.1-schnell` |
| `MODEL_T5` | Модель для генерации вопросов | `mrm8488/t5-base-finetuned-question-generation-ap` |


### Модели
#### Проект использует следующие предобученные модели:

 - **FLUX.1-schnell** - генерация изображений
 - **mrm8488/t5-base-finetuned-question-generation-ap** - генерация вопросов

### Интерфейс
#### Основные элементы:
 - **Поле ввода текста** - вставьте учебный материал
 - **Панель статистики** - показывает количество слов, предложений и символов
 - **Примеры текстов** - готовые примеры для тестирования
 - **Кнопка генерации** - запускает создание карточек
 - **Область результатов** - отображает сгенерированные карточки
 - **Панель экспорта** - скачивание карточек в ZIP архиве
